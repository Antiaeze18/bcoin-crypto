🚀 Bcoin – Web-Based Mining Simulation Platform

Bcoin is a web application that simulates a digital mining and reward system, inspired by modern crypto engagement platforms.
It is designed to demonstrate real-time data handling, user interaction, monetized task flows, and Firebase-powered backend services.

🌐 Live Demo
👉 https://bcoin-382da.web.app/

📌 Project Overview
Bcoin allows users to:
* Register and log in securely
* Simulate daily mining activity with cooldown logic
* Earn rewards through engagement tasks
* Track balances and transaction history
* Submit withdrawal requests (admin-reviewed)
* Interact with a structured referral system

This project was built to showcase full-stack web development using Firebase as a Backend-as-a-Service (BaaS).

⚙️ Development Note
This project was initially scaffolded using AI-assisted development tools to accelerate prototyping and structure.
All configuration, Firebase integration, feature logic, customization, testing, and deployment were implemented and managed manually.
The repository represents the maintained production codebase and ongoing development workflow.
